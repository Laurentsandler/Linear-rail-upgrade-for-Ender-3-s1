# March 4 7:37pm : coming up with idea and picking parts

I want to add a custom linear rail upgrade for my Ender 3 S1 for the X-axis or the tool head axis. 
After doing some research, I discovered that MG9H linear rails are the best, as they are about the same size as the cross beam that is already present on the machine and they're inexpensive. 
I measured my printer and figured out that I need a 350mm rail.
